mlprimitives.feature_extraction module¶

class mlprimitives.feature_extraction.CategoricalEncoder(max_labels=None, **kwargs)[source]¶

Bases: mlprimitives.feature_extraction.FeatureExtractor

Use the OneHotLabelEncoder only on categorical features.

NOTE: At the moment of this release, sklearn.preprocessing.data.CategoricalEncoder has not been released yet, this is why we write our own version of it.

>>> df = pd.DataFrame([
... {'a': 'a', 'b': 1, 'c': 1},
... {'a': 'a', 'b': 2, 'c': 2},
... {'a': 'b', 'b': 2, 'c': 1},
... ])
>>> ce = CategoricalEncoder()
>>> ce.fit_transform(df, categorical_features=['a', 'c'])
   b  a=a  a=b  c=1  c=2
0  1    1    0    1    0
1  2    1    0    0    1
2  2    0    1    1    0

class mlprimitives.feature_extraction.OneHotLabelEncoder(name=None, max_labels=None)[source]¶

Bases: object

Combination of LabelEncoder + OneHotEncoder.

>>> df = pd.DataFrame([
... {'a': 'a', 'b': 1, 'c': 1},
... {'a': 'a', 'b': 2, 'c': 2},
... {'a': 'b', 'b': 2, 'c': 1},
... ])
>>> OneHotLabelEncoder().fit_transform(df.a)
   a=a  a=b
0    1    0
1    1    0
2    0    1
>>> OneHotLabelEncoder(max_labels=1).fit_transform(df.a)
   a=a
0    1
1    1
2    0
>>> OneHotLabelEncoder(name='a_name').fit_transform(df.a)
   a_name=a  a_name=b
0         1         0
1         1         0
2         0         1
