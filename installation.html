Installation¶

Stable release¶

To install MLPrimitives, run this command in your terminal:

$ pip install mlprimitives

This is the preferred method to install MLPrimitives, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for MLPrimitives can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/HDI-Project/MLPrimitives

Or download the tarball:

$ curl  -OL https://github.com/HDI-Project/MLPrimitives/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
